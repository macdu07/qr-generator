# QR Generator by Naikka

A modern QR Code generator with advanced customization options for WordPress.

## Description

QR Generator by Naikka is a lightweight and powerful WordPress plugin that allows you to embed a fully customizable QR Code generator on any page or post. Users can generate QR codes for URLs, text, emails, and WhatsApp messages, with options to customize colors, shapes, and logos.

## Features

- **Multiple Content Types**: Generate QR codes for Links, Text, E-mail, and WhatsApp.
- **Customization**:
  - Change background and foreground colors.
  - Customize dot styles (Square, Dots, Rounded, Classy, etc.).
  - Customize corner markers.
- **Logo Support**: Upload and embed custom logos in the center of the QR code.
- **Download Options**: Download generated QR codes as PNG or SVG.
- **Live Preview**: See changes instantly as you customize.

## Installation

1. Upload the `qr-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[qr_generator]` in any page or post to display the generator.

## Usage

Simply add the following shortcode to any page or post where you want the QR generator to appear:

```shortcode
[qr_generator]
```
